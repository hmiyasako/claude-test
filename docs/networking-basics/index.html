ネットワークの基礎概念をカテゴリー別に理解する

📚

OSI参照モデル

Open Systems Interconnection

ネットワーク通信を7つの層に分けて標準化した概念モデル。異なるシステム間の通信を可能にします。

7層モデル 標準化
🔗

TCP/IPモデル

Transmission Control Protocol/Internet Protocol

インターネットで実際に使用されている4層のプロトコルスタック。現代のネットワーク通信の基盤です。

4層モデル インターネット標準
📡

パケット通信

Packet Switching

データを小さなパケットに分割して送信する通信方式。効率的なネットワーク利用を実現します。

データ分割 効率的通信
🔄

TCP

Transmission Control Protocol

信頼性の高いコネクション型プロトコル。データの到着順序と完全性を保証します。

信頼性 コネクション型
⚡

UDP

User Datagram Protocol

高速なコネクションレス型プロトコル。リアルタイム通信やストリーミングに最適です。

高速 コネクションレス
🌍

HTTP/HTTPS

HyperText Transfer Protocol (Secure)

Webページの転送に使用されるプロトコル。HTTPSはSSL/TLSによる暗号化を追加。

Web通信 暗号化
📧

SMTP/POP/IMAP

メールプロトコル

電子メールの送受信に使用されるプロトコル群。SMTPは送信、POP/IMAPは受信に使用。

メール送信 メール受信
🔢

IPv4

Internet Protocol version 4

32ビットのIPアドレスを使用。約43億個のアドレスを提供しますが、枯渇が問題となっています。

32ビット 現行標準
🔠

IPv6

Internet Protocol version 6

128ビットのIPアドレスを使用。事実上無限のアドレス空間を提供する次世代プロトコル。

128ビット 次世代
🎭

サブネット

Subnet / CIDR

大きなネットワークを小さなネットワークに分割する技術。効率的なIPアドレス管理を実現。

ネットワーク分割 CIDR表記
🏠

MACアドレス

Media Access Control Address

ネットワーク機器に割り当てられた一意の物理アドレス。48ビットで表現されます。

物理アドレス 一意識別子
🔀

ルーター

Router

異なるネットワーク間でパケットを転送するL3デバイス。最適な経路を選択してデータを届けます。

L3デバイス 経路制御
🔌

スイッチ

Switch

同一ネットワーク内でフレームを転送するL2デバイス。MACアドレスを学習して効率的に通信。

L2デバイス フレーム転送
📶

アクセスポイント

Wireless Access Point

無線LANの基地局として機能するデバイス。有線ネットワークと無線デバイスを接続します。

無線LAN Wi-Fi
⚖️

ロードバランサー

Load Balancer

複数のサーバーにトラフィックを分散させるデバイス。高可用性とスケーラビリティを実現。

負荷分散 高可用性
📖

DNS

Domain Name System

ドメイン名をIPアドレスに変換するシステム。インターネットの電話帳のような役割を果たします。

名前解決 ドメイン管理
🎫

DHCP

Dynamic Host Configuration Protocol

ネットワーク設定を自動的に割り当てるプロトコル。IPアドレス、サブネット、ゲートウェイなどを配布。

自動設定 アドレス配布
🔄

NAT

Network Address Translation

プライベートIPアドレスをグローバルIPアドレスに変換。IPv4アドレスの節約に貢献します。

アドレス変換 IPv4節約
🖥️

プロキシ

Proxy Server

クライアントとサーバーの間に立つ中継サーバー。キャッシュ、フィルタリング、匿名化などを提供。

中継サーバー キャッシュ
🛡️

ファイアウォール

Firewall

ネットワークトラフィックを監視・制御するセキュリティシステム。不正アクセスからネットワークを保護。

アクセス制御 防御
🔒

VPN

Virtual Private Network

暗号化されたトンネルを通じて安全な通信を実現。リモートアクセスや拠点間接続に使用。

暗号化トンネル リモートアクセス
🔐

SSL/TLS

Secure Sockets Layer / Transport Layer Security

通信を暗号化するプロトコル。Webサイトのhttps、メール、VPNなどで広く使用されています。

暗号化 証明書
🕵️

IDS/IPS

Intrusion Detection/Prevention System

ネットワーク上の不正活動を検知・防止するシステム。サイバー攻撃からの保護を提供。

侵入検知 侵入防止
