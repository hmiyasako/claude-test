🚀 AWS主要サービス ビジュアルガイド

Amazon Web Servicesの主要サービスをカテゴリー別に理解する

☁️

EC2

Elastic Compute Cloud

仮想サーバーをクラウド上で起動・管理できるサービス。スケーラブルなコンピューティング能力を提供します。

仮想サーバー スケーラブル
⚡

Lambda

AWS Lambda

サーバーレスコンピューティングサービス。サーバー管理なしでコードを実行できます。イベント駆動型の処理に最適。

サーバーレス イベント駆動
📦

ECS

Elastic Container Service

Dockerコンテナを簡単にデプロイ、管理、スケーリングできるコンテナオーケストレーションサービス。

コンテナ オーケストレーション
🗄️

S3

Simple Storage Service

高い耐久性を持つオブジェクトストレージサービス。画像、動画、バックアップなど、あらゆるデータを保存できます。

オブジェクトストレージ 耐久性99.999999999%
💾

EBS

Elastic Block Store

EC2インスタンス用の永続的なブロックストレージ。高性能で低レイテンシーのストレージボリュームを提供。

ブロックストレージ 高性能
🧊

Glacier

Amazon S3 Glacier

長期アーカイブ用の低コストストレージサービス。アクセス頻度の低いデータの保存に最適。

アーカイブ 低コスト
🗃️

RDS

Relational Database Service

マネージド型リレーショナルデータベースサービス。MySQL、PostgreSQL、Oracle、SQL Serverなどに対応。

リレーショナルDB マネージド
⚡

DynamoDB

Amazon DynamoDB

フルマネージドNoSQLデータベースサービス。高速で予測可能なパフォーマンスと自動スケーリングを提供。

NoSQL 高速
🌟

Aurora

Amazon Aurora

MySQL・PostgreSQL互換のクラウド最適化されたリレーショナルデータベース。高性能と高可用性を実現。

高性能 MySQL/PostgreSQL互換
🌐

VPC

Virtual Private Cloud

AWS内に論理的に分離されたプライベートネットワークを構築。完全にコントロール可能な仮想ネットワーク環境。

プライベートネットワーク 隔離
🚀

CloudFront

Amazon CloudFront

高速コンテンツ配信ネットワーク（CDN）サービス。世界中のエッジロケーションからコンテンツを配信。

CDN グローバル配信
🗺️

Route 53

Amazon Route 53

高可用性でスケーラブルなDNSウェブサービス。ドメイン名を管理し、ヘルスチェック機能も提供。

DNS 高可用性
🔐

IAM

Identity and Access Management

AWSリソースへのアクセスを安全に管理。ユーザー、グループ、ロールを作成し、きめ細かいアクセス制御を実現。

アクセス管理 認証・認可
👤

Cognito

Amazon Cognito

ウェブ・モバイルアプリにユーザー認証、認可、管理機能を追加。ソーシャルログインにも対応。

ユーザー認証 ソーシャルログイン
🔑

KMS

Key Management Service

暗号化キーの作成と管理を簡単に実行。データの暗号化と復号化を安全に行えます。

暗号化 キー管理
📊

CloudWatch

Amazon CloudWatch

AWSリソースとアプリケーションの監視サービス。メトリクス収集、ログ管理、アラーム設定が可能。

モニタリング ログ管理
🔬

Athena

Amazon Athena

標準SQLを使用してS3内のデータを直接分析。サーバーレスのインタラクティブなクエリサービス。

SQLクエリ サーバーレス
📈

QuickSight

Amazon QuickSight

高速なビジネスインテリジェンスサービス。データを可視化し、インサイトを簡単に発見できます。

BI データ可視化
💻

CodeCommit

AWS CodeCommit

フルマネージドなプライベートGitリポジトリホスティングサービス。安全にコードを保存・管理できます。

Gitリポジトリ ソース管理
🔨

CodeBuild

AWS CodeBuild

フルマネージドなビルドサービス。ソースコードをコンパイルし、テストを実行し、デプロイ可能なパッケージを作成。

CI/CD ビルド自動化
🚢

CodeDeploy

AWS CodeDeploy

アプリケーションのデプロイを自動化するサービス。EC2、Lambda、オンプレミスサーバーに対応。

デプロイ自動化 CI/CD
